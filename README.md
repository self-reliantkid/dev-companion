# dev-companion

An AI-powered documentation and test generation tool built with IBM Bob.
Point it at a Python codebase and it automatically generates docs, tests,
and detects when code changes make them stale.

Built for the IBM Bob Hackathon · May 2026

---

## What it does

| Command | What Bob does |
|---|---|
| `/gendoc` | Generates Google-style docstrings and a markdown doc file |
| `/gentest` | Generates a full pytest test suite with edge cases and error cases |
| `/sync` | Detects stale docs and tests after code changes and flags them |

---

## How it works

1. Open the repo in IBM Bob IDE
2. Run `/gendoc src/task_service.py` — Bob reads the code and generates
   Google-style docstrings inline and a markdown file in `docs/`
3. Run `/gentest src/task_service.py` — Bob generates 68 pytest tests
   covering happy paths, edge cases, and all exceptions
4. Change a function signature without updating docs or tests
5. Run `/sync src/task_service.py` — Bob detects the mismatch,
   flags the stale item with file, line number, and reason
